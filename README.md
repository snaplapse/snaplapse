# SnapLapse

## Setup
Install pyenv
https://github.com/pyenv/pyenv

Install python
```sh
pyenv install 3.10.4
```

Setup pipenv
```sh
# macOS
brew install pipenv

# Windows

# Install dependencies
pipenv --python 3.10.4 install
```

## Run
```sh
# Launch virtual environment
pipenv shell

# Run the server
python manage.py runserver
```
